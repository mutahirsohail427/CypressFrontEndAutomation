Automation: Cypress + Javascript
Scneario: Automation of basic Checkout flow for an ecommerce website
Steps:
1- Open the browser
2- Search for items
3- Add to cart
4- Validate total Sum of items in cart to total amount
5- Complete Checkout process
6- Assert the success of Checkout flow

Execution: 
1- Make sure to have an IDE, Node and everything setup
2- Open the code and go to Terminal
3- Go to project repo in the Terminal
4- Run the command node_modules/.bin/cypress open
